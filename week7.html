get_probability <- function(N, K, n, k) { choose(K, k) * choose(N - K, n - k) / choose(N, n) }

N <- 10 K <- 6 n <- 5 possible_k <- 0:n

probabilities <- sapply(possible_k, function(k) get_probability(N, K, n, k))

A stationary time series has constant mean and variance over time. Its essential for: - Forecasting - Valid modeling - Avoiding spurious regression

8.11.1.1 Unit Root: Augmented Dickey-Fuller (ADF) Test

library(tseries) set.seed(42) x <- cumsum(rnorm(100)) # non-stationary random walk plot.ts(x, main = “Simulated Random Walk”)

adf.test(x) # Likely non-stationary (p > 0.05) 7.2 Detrending Time Series
